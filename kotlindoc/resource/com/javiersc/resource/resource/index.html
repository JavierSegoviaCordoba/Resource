Resource

sealed class Resource<S, E> : Any

This class lets wrap any thing and add an associated state: Loading, Success and Error.

inline fun <S1, E, S2, NS> Resource<S1, E>.combine(resource: Resource<S2, E>, success: (S1, S2)->NS): Resource<NS, E>

Combine two Resource into one. Both resources has the same error type. Success happens when both are success Loading happens when one is Loading and the another is Loading or Success If only one resource is error, it will be returned as error If both resources are errors, this will be returned as error
